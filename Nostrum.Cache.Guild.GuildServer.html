    get_value(search_criteria, key) 
    get_value(search_criteria, key :: atom) ::
  {:error, reason :: atom} |
  {:ok, term}

    Retrieves a value from a guild in the cache.

    This method should be preferred over the Nostrum.Cache.Guild.GuildServer.get/1 method in the event that you need only one kv pair from the guild.

    Parameters

    • search_criteria - The criteria used to search. See search_criteria/0 for more info.
    • key - The key to search for. See Nostrum.Struct.Guild for available keys.

    Example 

    case Nostrum.Cache.Guild.Guildserver.get_value([id: 69696969696969], :name) do
  {:ok, sick_guild_name} -> "#{sick_guild_name} is sick"
  {:error, reason} -> "Whatchu' searchin' for fam?"
end
